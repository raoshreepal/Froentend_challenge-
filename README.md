
# JavaScript Quiz App

A simple and interactive **quiz application** built using HTML, CSS, and JavaScript.  
It presents 20 multiple-choice questions and shows the score at the end.

## 📌 Features
- 20 multiple-choice quiz questions.
- Interactive UI for answering questions.
- Score calculation at the end of the quiz.
- Responsive design for mobile and desktop.

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Run
1. Download or clone this repository:
   
   git clone hhttps://github.com/raoshreepal/Froentend_challenge-.git


2. Open `index.html` in your browser.
